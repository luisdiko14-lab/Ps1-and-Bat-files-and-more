# Luis Toolkit

**Luis Toolkit** is a collection of fun, practical, and interactive Windows scripts written in **PowerShell (.ps1)** and **Batch (.bat)**. It includes tools for system info, animations, password generation, website monitoring, cleanup, and more — all accessible via easy-to-use menus.

---

## Features

### **Batch Scripts**
- **Matrix Rain Effect** – Classic green “Matrix” console effect.  
- **Random Password Generator** – Generates a strong 12-character password.  
- **Open Favorite Websites** – Opens multiple favorite websites at once.  
- **System Cleanup** – Deletes temporary files to free up disk space.  
- **Fake Virus Prank** – Harmless looping warning for fun.

### **PowerShell Scripts**
- **Progress Bar** – Animated colorful progress bar in console.  
- **PC Info Dashboard** – Displays CPU, RAM, Disk space, and OS info.  
- **Website Ping Monitor** – Continuously checks if a website is online.  
- **Pop-up Reminder** – Shows Windows GUI reminder.  
- **Animated Text / Marquee** – Rotating text animation in console.

### **Interactive Toolkits**
- **LuisToolkit.bat** – Batch menu to run selected scripts easily.  
- **LuisToolkit.ps1** – PowerShell menu version with progress bar, dashboard, ping monitor, and reminders.
