# Python Mega Bot (Generated)

## Quick start

1. `cd python_bot`
2. `pip install -r requirements.txt`
3. Put your bot token into `.env` (TOKEN=)
4. `python main.py`

- 20+ General commands (info, fun, util)
- 15+ Moderation commands (permissions enforced)
- 10+ Music commands (structure & placeholders)
- Modular cogs (cogs/general.py, cogs/moderation.py, cogs/music.py)
